# Breakout

# Objetivo del juego
El objetivo del juego es destruir todos los bloques en la pantalla usando la pelota y el paddle.  
La pelota rebota en el paddle y en los bordes del canvas. Se pierde una vida si la pelota pasa por el borde inferior.


# Cómo jugar
1. Primero tienes que abrir el archivo index.html (Click derecho - Open with live server).
2. Luego tienes que ingresar el número de filas y columnas de bloques que quieres que haya en la ronda.
3. El juego comienza cuando mueves la pelota.
4. Usa el paddle para evitar que la pelota caiga por la parte de abajo.


# Controles
- Mover paddle a la izquierda: flecha izquierda  
- Mover paddle a la derecha: flecha derecha  

# Nota: El paddle se mueve solo horizontalmente.

# Reglas del juego
- Tienes 3 vidas.  
- Cada vez que la pelota pasa por la parte de abajo, pierdes una vida.  
- La pelota aumenta de velocidad cada 5 bloques destruidos.  
- Ganarás el juego si destruyes todos los bloques.  
- Si pierdes todas las vidas, aparece GAME OVER, y tendrás que empezar el juego otra vez.

# Funcionalidades extra
- Los bloques tienen un degradé de colores de azul oscuro → gris → blanco.  
- El juego muestra un contador de tiempo en segundos y milisegundos para que puedas ver en ccuanto tiempo acabas la ronda y superar tu record en la próxima.  
- Se muestra en pantalla el número de bloques destruidos.
